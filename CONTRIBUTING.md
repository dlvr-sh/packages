# Contributing

Thank you for improving the dlvr.sh clients and integrations.

1. Open an issue for substantial API or behavior changes before implementing them.
2. Keep credentials out of fixtures, logs, screenshots, and commits.
3. Add tests for protocol, validation, and error-handling changes.
4. Run `bun run verify` before opening a pull request.
5. Use English for public interfaces and documentation.

Contributions are accepted under the repository's MIT license.
